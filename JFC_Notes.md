
# 📚 Guía Rápida y Referencia del Proyecto

## 🖥️ Frontend
- **Framework:** [React](https://react.dev/)
- **Renderizado:** [Canvas API (HTML5)](https://developer.mozilla.org/es/docs/Web/API/Canvas_API)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/docs/installation/using-vite) (responsive)

## 🛠️ Backend
- **Framework:** [Express](https://expressjs.com) (Node.js)
- **Tiempo real:** [Socket.io](https://socket.io/docs/v4/)

## 🗄️ Base de Datos
- **ODM:** [Mongoose](https://mongoosejs.com/docs/guide.html)

## 🐳 Docker y Contenerización

## 🏗️ Estructura y Buenas Prácticas

### 👤 Gestión de Usuarios y Seguridad
- **Registro/Login:** Validación de inputs (frontend y backend)
- **Privacidad:** Añadir páginas de *Privacy Policy* y *Terms of Service* accesibles desde el footer

### 🎮 Game Loop
- **Servidor:** Calcula posiciones, colisiones, daños, etc.
- **Cliente:** Envía inputs y recibe el estado actual para renderizar

### 🚀 Despliegue y Roles
- **Contenedores:** Todo debe correr en **Docker** (Frontend, Backend y DB)
- **Roles de equipo:** PO, PM, Tech Lead y Dev(s)
